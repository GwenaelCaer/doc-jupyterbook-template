# doc-jupyterbook-template

## Install the project

```bash
# install jupyterbook
pip install -r requirements.txt

# install libraries needed to execute your code  
pip install -r .binder/requirements.txt
```

## How to use

Run a local server:

```bash
jupyter book start
```
